## 📦 安装与更新

推荐使用 git 进行安装，以方便后续升级。在[Yunzai-Bot](https://gitee.com/Le-niao/Yunzai-Bot)根目录打开终端，运行如下命令进行安装。

```base
git clone https://gitee.com/xianxincoder/xianxin-plugin.git ./plugins/xianxin-plugin/
```
